## 判断

1. `dewrap_join` 新 chunk base 的分块数量和正文长度已经非常接近原文；这一步值得继续。
2. 摘要长度单位需要固定：如果原文 `Summary_len` 本质更接近中文摘要字符 proxy，当前 prompt 已经接近；如果原文也用 GLM tokenizer 计摘要，当前摘要过长，需要把 prompt 缩短。
3. 50 家样本下 Panel B 主要看方向、显著性和低/高评分中位数关系；Panel C/D 用 firm-cluster 与 Spearman 同时核对。
4. 下一步建议做 100 家确认或直接制定全样本重跑方案；前提是先固定 `Summary_len` 到底采用旧摘要 proxy 还是 GLM-4 summary tokens。
